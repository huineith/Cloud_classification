# Project to create a simple ML Cloud classifaction program 

## data 
The data set consist of 8000 ground based cloud images taken in Tianjin China 
classified into 7 categories 
- cumulus
- altocomulus
- cirrus
- clear sky
- stratocumulus
- columulonibus
- mixed clouds
